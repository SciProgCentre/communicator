class IncompatibleSpecsException(functionName: String, localSpec: String, remoteSpec: String) : RemoteCallException
Content copied to clipboard

This exception is thrown on the client if its coders have a different identities than the coders on the remote server. This exception can also be thrown on the worker, if it tries to connect to a proxy, and proxy already has a worker for the same function but with different coder identities. Spec strings are the strings received from FunctionSpec.toString method, which uses Coder.toString.
